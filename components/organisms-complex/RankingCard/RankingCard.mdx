# RankingCard

This component makes it easier to render a basic Ranking card in a McGonagall flow. The collapsed state of this card uses the Simple Summary component. This card is typically used to rank different items against eachother.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```tsx
import RankingCard from '@unitedincome/components/dist/RankingCard';
```

```tsx
import {RankingCard} from '@unitedincome/components';
```

### Using in a McGonagall Flow

You can use the `SliderCard` component in a McGonagall flow using the following pattern.

```tsx
export const renderCard = (props, state) => {
  return (
    <RankingCard
      {...prop}
      config={{
        name: 'yesNo',
        items: [
          {
            label: 'Montezuma',
            secondaryLabel: '$8,345',
            id: 'montezuma',
            movable: false,
          },
          {
            label: 'Pica',
            secondaryLabel: '$1,000',
            id: 'pica',
            movable: true,
            focus: true,
          },
          {
            label: 'Pixie',
            secondaryLabel: '$13,000',
            id: 'pixie',
            movable: false,
          },
        ],
      }}
    />
  );
};
```
