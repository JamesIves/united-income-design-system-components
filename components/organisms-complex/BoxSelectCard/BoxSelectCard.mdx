# BoxSelectCard

This component makes it easier to render a basic box select card in McGonagall flows. This card is typically used on flow entry screens.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import BoxSelectCard from '@unitedincome/components/dist/BoxSelectCard';
```

```javascript
import {BoxSelectCard} from '@unitedincome/components';
```

### Using in a McGonagall Flow

You can use the `BoxSelectCard` component in a McGonagall flow using the following pattern.

```javascript
export const renderCard = (props, state) => {
  return (
    <BoxSelectCard
      {...props}
      config={{
        name: 'selectBtn',
        value: 'yes',
        options: [
          {
            label: 'Essentials',
            description: 'Lets talk about essentials',
            icon: CarCircleIllustration,
            value: 'yes',
          },
          {
            label: 'Lifestyle',
            description: 'Lets talk about lifestyle',
            icon: LampCircleIllustration,
            value: 'no',
          },
          {
            label: "I don't know",
            description: 'I am really not sure honestly',
            icon: CakeCircleIllustration,
            showCheck: true,
            disabled: true,
            value: 'idk',
          },
        ],
      }}
    />
  );
};
```
