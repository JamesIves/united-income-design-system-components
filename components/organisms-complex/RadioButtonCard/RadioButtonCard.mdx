# RadioButtonCard

This component makes it easier to render a basic radio button card in a McGonagall flow. The collapsed state of this card uses the Simple Summary component. This card is typically used when there's only one valid answer to a question.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```tsx
import RadioButtonCard from '@unitedincome/components/dist/RadioButtonCard';
```

```tsx
import {RadioButtonCard} from '@unitedincome/components';
```

### Using in a McGonagall Flow

You can use the `RadioButtonCard` component in a McGonagall flow using the following pattern.

```tsx
export const renderCard = (props, state) => {
  return (
    <RadioButtonCard
      {...props}
      config={{
        value: 'yes',
        name: 'yesNo',
        options: [
          {
            label: 'Yes',
            value: 'yes',
          },
          {
            label: 'No',
            value: 'no',
          },
          {
            label: "I don't know",
            value: 'idk',
            disabled: true,
          },
        ],
      }}
    />
  );
};
```
