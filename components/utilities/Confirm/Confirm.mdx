# Confirm

Helper function which allows you to call a confirmation modal similar to how you use `window.confirm`. Renders a `Modal` component when called.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import Confirm from '@unitedincome/components/dist/Confirm';
```

```javascript
import {Confirm} from '@unitedincome/components';
```

## Example ðŸš€

```javascript
Confirm({
  name: 'Cancel Changes',
  title: 'Are you sure?',
  description: `If you leave this card without hitting "Save Changes," all of your changes will be lost.`,
  okLabel: 'Confirm',
  cancelLabel: 'Cancel',
})
  .then(() => console.log('User clicked confirm'))
  .catch(() => console.log('User clicked cancel'));
```
