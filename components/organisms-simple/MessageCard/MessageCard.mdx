# MessageCard

This component renders a message card for the Hogwarts/McGonagall flow used for intros or to explain new questions.

You can use this component using one of the following import patterns.


```javascript

```

### Using in a McGonagall Flow

You can use the `MessageCard` component in a McGonagall flow using the following pattern.

```javascript
export const renderCard = (props, state) => {
  return (
    <MessageCard
      {...props}
      featureImage={
        <img
          alt="example"
          src="https://app.unitedincome.com/assets/images/pencil.svg"
          style={{height: 100, width: 100}}
        />
      }
    />
  );
};
```

### Collapsed

You can collapse the card using the `isCollapsed` prop. The McGonagall framework will automatically hand this prop down when it's not the current card.
