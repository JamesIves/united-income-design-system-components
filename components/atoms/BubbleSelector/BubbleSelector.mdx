# BubbleSelector

The BubbleSelector component renders a horizontal item selector in a bubble style.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import BubbleSelector from '@unitedincome/components/dist/BubbleSelector';
```

```javascript
import {BubbleSelector} from '@unitedincome/components';
```

### Bubble Colors

You can change each bubbles color by providing the option with a `color` property.

<Preview>
  <BubbleSelector

      {value: 'montezuma2'},
      {value: 'montezuma3', color: 'Red'},
    ]}
    value="montezuma3"
    onChange={(value) => true}
  />
</Preview>

<BubbleSelector

### Adjusting the Width

If you'd like to change the width of the bubbles, you can provide the component with a `fixedWidth` prop.
