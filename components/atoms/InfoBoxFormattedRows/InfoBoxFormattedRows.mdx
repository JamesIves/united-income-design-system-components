# InfoBoxFormattedRows

The InfoBoxFormattedRows component renders your `(label, value)` pairs in neatly formatted rows. It
is a sub-component of the shell [InfoBox](/?path=/docs/atoms-infobox--default) component.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import InfoBoxFormattedRows from '@unitedincome/components/dist/InfoBoxFormattedRows';
```

```javascript
import {InfoBoxFormattedRows} from '@unitedincome/components';
```
