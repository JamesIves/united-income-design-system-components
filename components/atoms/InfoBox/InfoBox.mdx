# InfoBox

The InfoBox component renders a card which can be used to display custom
information to a user in a convenient format.

## Styling Content as Formatted Rows ðŸ‘”

While InfoBox gives you the flexibility to pass in any custom content and/or footer and have them be rendered
to your speicifications, it also has built-in logic that allows you to create neatly formatted rows of
`(label, value)` pairs. The [InfoBoxFormattedRows](/?path=/docs/atoms-infoboxformattedrows--default) component
allows you to easily pass in your data as an array of JS objects and have it render neatly within the InfoBox component.
Additionally, it should be noted that when you add footers we automatically create a light gray `<hr />` divider between
the card content and the footer.
