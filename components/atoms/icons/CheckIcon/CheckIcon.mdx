# CheckIcon

The CheckIcon component creates a checkmark icon using inline SVG.

You can use this component using one of the following import patterns.

```javascript

```

```javascript

```

### Changing the Color

You can adjust the color of the icon using the `fill` prop. This prop accepts common color names and hex values.

<Preview>
  <CheckIcon height="3rem" width="3rem" fill="#ababab" />
</Preview>

```javascript
<CheckIcon height="3rem" width="3rem" fill="#ababab" />
```

### Removing Background

If you'd like to render the check without the background you can toggle `removeBackground` to `true.`

<Preview>
  <CheckIcon height="3rem" width="3rem" removeBackground />
</Preview>

```javascript
<CheckIcon height="3rem" width="3rem" removeBackground />
```
