### Reccomended Position

Depending on the question you may wish to show a reccomended value on the slider. This can be achieved by using the `tooltipStickyPosition` and `tooltipStickyVariant` props.

<Preview>
  <Slider
    value={3}
    leftAnnotate="Lower Risk"
    rightAnnotate="Higher Risk"
    tooltip="You can pass any arbitrary node into the tooltip to style it how you'd Like"
    tooltipStickyPosition={5}
    tooltipStickyVariant="green"
  />
</Preview>

```javascript
<Slider
  value={3}
  leftAnnotate="Lower Risk"
  rightAnnotate="Higher Risk"
  tooltip="You can pass any arbitrary node into the tooltip to style it how you'd Like"
  tooltipStickyPosition={5}
  tooltipStickyVariant="green"
/>
```
