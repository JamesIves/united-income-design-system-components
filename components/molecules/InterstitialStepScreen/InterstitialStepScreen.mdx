# InterstitialStepScreen

Displays a full page interstitial step screen. This component is commonly used to inform/guide the user regarding the next steps they should take.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import InterstitialStepScreen from '@unitedincome/components/dist/InterstitialStepScreen';
```

```javascript
import {InterstitialStepScreen} from '@unitedincome/components';
```
