# TabbedTile

Creates a Tile component with a series of tab navigation anchors at the top of it. The tabbed tile is optimized for 2-3 navigation items.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import TabbedTile from '@unitedincome/components/dist/TabbedTile';
```

```javascript
import {TabbedTile} from '@unitedincome/components';
```

### Dark Mode
