# BubbleNav

The BubbleNav component renders a responsive navigation selector in a solid color bubble style. This component dynamically switches between the `BubbleSelector` and `BubbleDropdown` components.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import BubbleNav from '@unitedincome/components/dist/BubbleNav';
```

```javascript
import {BubbleNav} from '@unitedincome/components';
```

### Bubble Colors

You can change each bubbles color by providing the option with a `color` property.

<Preview>
  <BubbleNav
    options={[

      {value: 'montezuma2'},
      {value: 'montezuma3', color: 'Red'},
    ]}
    value="montezuma3"
    onChange={(value) => true}
  />
</Preview>

```javascript
<BubbleNav
  options={[

    {value: 'montezuma3', color: 'Red'},

```

### Adjusting the Width

If you'd like to change the width of the bubbles, you can provide the component with a `fixedWidth` prop.

<Preview>

      {value: 'montezuma2'},
      {value: 'montezuma3', color: 'Red'},
    ]}
    value="montezuma3"
    onChange={(value) => true}
  />
</Preview>
