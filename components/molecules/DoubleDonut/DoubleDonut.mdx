# DoubleDonut

Renders the Double Donut chart component. This component can be used to display a breakdown of items by percent.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import DoubleDonut from '@unitedincome/components/dist/DoubleDonut';
```

```javascript
import {DoubleDonut} from '@unitedincome/components';
```
