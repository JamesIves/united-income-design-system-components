# MultiColumnTile

Creates a MultiColumn tile that represents data in a single column for small devices and two columns for medium to large devices. This component also provides pagination.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import MultiColumnTile from '@unitedincome/components/dist/MultiColumnTile';
```

```javascript
import {MultiColumnTile} from '@unitedincome/components';
```
