# PaginatedTile

Creates a Tile component that enables pagination. This allows the tile to render different views on separate pages.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import PaginatedTile from '@unitedincome/components/dist/PaginatedTile';
```

```javascript
import {PaginatedTile} from '@unitedincome/components';
```

      <ul style={{fontSize: '1.4rem', color: 'inherit'}}>

        <li>Tuesday</li>
      </ul>,
      <ul style={{fontSize: '1.4rem', color: 'inherit'}}>
        <li>Wednesday</li>

```javascript
<PaginatedTile
  pages={[
    <ul style={{fontSize: '1.4rem', color: 'inherit'}}>
      <li>Monday</li>
      <li>Tuesday</li>
    </ul>,
    <ul style={{fontSize: '1.4rem', color: 'inherit'}}>
      <li>Wednesday</li>
      <li>Thursday</li>
    </ul>,

/>
```
