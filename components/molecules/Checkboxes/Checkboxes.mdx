# Checkboxes

The Checkboxes component renders a group of HTML select buttons with a number of customization options.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import Checkboxes from '@unitedincome/components/dist/Checkboxes';
```

```javascript
import {Checkboxes} from '@unitedincome/components';
```

### Question Followup

If your answer requires a followup input or response, you can utilize the `followup` key in the `options` prop.

### Secondary label

Depending on the scenario you may require the use of a secondary label that appears to the far right of the option. You can pass `secondaryLabel` to each option that requires it. This option works best with the `table` prop, as demonstrated below.

<Preview>
  <Checkboxes
    table={true}
    options={[
      {
        label: 'Montezuma is the worlds best cat, this is guaranteed.',
        value: 'montezuma1',
        disabled: true,
        secondaryLabel: '$200',
      },
      {
        label: 'Montezuma is a really good ginger boi',
        value: 'montezuma2',
        secondaryLabel: '$300',
      },
      {
        label: 'Some people say he is the best cat ever',
        value: 'montezuma3',
        secondaryLabel: '$400',
      },
    ]}
    value="montezuma3"
  />
</Preview>
