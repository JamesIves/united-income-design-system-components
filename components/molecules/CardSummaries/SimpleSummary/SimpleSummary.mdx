# SimpleSummary

This component renders a simple summary to be used with a card component when it is collapsed. This summary component should be used for simpler questions like radio button or checkbox questions.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript

```

```javascript
import {SimpleSummary} from '@unitedincome/components';
```
