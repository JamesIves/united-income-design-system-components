# FormSummary

This component renders a summary for McGonnagall question cards. It's primarily used to represent user choices on a specific screen after they are done making their selection.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import FormSummary from '@unitedincome/components/dist/FormSummary';
```

```javascript
import {FormSummary} from '@unitedincome/components';
```
