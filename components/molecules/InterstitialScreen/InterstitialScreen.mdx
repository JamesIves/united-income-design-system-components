# InterstitialScreen

Displays an full page interstitial screen. This component is commonly used to inform/guide the user regarding the next steps they should take.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import InterstitialScreen from '@unitedincome/components/dist/InterstitialScreen';
```

```javascript
import {InterstitialScreen} from '@unitedincome/components';
```
